   - Create new project

## 📝 Basic Usage

```latex
\documentclass[theme-classic]{gamifiedcv}

\begin{document}
% Your info
\name{YOUR NAME}
\jobtitle{Your Job Title}
\level{Your Level}
...

% Make the header
\makeheader

% Add your content
...

\end{document}
```
